Functionality
-------------

- Ability to add and manage clients

- Import and export
	- Import information for house sale in one place
		- Excel
	- Export in Excel
	- Export Birthdays for a certain month
	- Event Tracking / Communicate

- Default Categories
	- Name
	- Birthday
	- Current Address
	- Buy / Sell
		- Address
	- Source
	- Status
	- Last Contact Date
	- Notes

- New Categories
	- Title
	- Description
	- Type
	- Dropdown Options
	- Editable on homescreen

- Sort
	- Date
	- All buyers / sellers
	- Source (enum)

- Edit any field at the same time

- Print address labels

- HAVE TO HAVE at least one client name and one form of communication (phone # or email)

- First and Last name for each client

- Type of Transaction - Buyer, seller, landlord, tennat, prospect, unspecified
